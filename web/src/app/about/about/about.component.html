About
The purpose of this dissertation project is to provide a user-friendly way to interact with multiple fully-trained variants of the 'Astronet' machine learning model. Astroweb will seemlessly download the astronomical data, validate the given parameters against the correct model, and return the results along with any relavent information about the star we may already know, including existing exoplanets.
